MicroservicesDemo

This project demo showcases a microservices architecture built with API Gateway (Ocelot), Service Registry and Discovery (Consul), RabbitMQ, Swagger, PostgreSQL with EF Core, Redis, OpenTelemetry, Clean Architecture, SOLID principles, and unit tests.

Testing procedures

Step1:Install [Docker Desktop](https://docs.docker.com/desktop/setup/install/windows-install/) and [Postman](https://www.postman.com/downloads/)

Step2:Download [Postman json file](https://github.com/shaohuamo/MicroservicesDemo/blob/master/MicroservicesDemo.postman_collection.json)

Step3:Import Postman json file into Postman(refer to step6)

Step4:Download [docker-compose.yml](https://github.com/shaohuamo/MicroservicesDemo/blob/master/docker/debug/docker-compose.yml)

Step5:Locate the path where you stored the .yml file and run command like this : `docker compose -f docker/debug/docker-compose.yml -f docker/debug/docker-compose.override.yml up`  (refer to step6)
  
Step6:Download TestCase.xlsx, click "view raw" after click [TestCase](https://github.com/shaohuamo/MicroservicesDemo/blob/master/TestCase.xlsx)
